## 🐓JURAM's Comment
첫번째 개인프로젝트로 새로운 지코바 치킨의 웹사이트를 직접 기획하여 구현하였습니다.

express로 메인페이지와 서브페이지를 연결하고 mysql, multer를 사용하여 게시판 CRUD 기능을 처음으로 만들어보았습니다.

반응형 구현까지 완료되었습니다.

(현재는 메인페이지만 접속이 가능합니다.)
